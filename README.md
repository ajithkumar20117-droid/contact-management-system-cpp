📱 Contact Management System (C++)

📌 Project Description

This project is a simple Contact Management System implemented using C++. It allows users to store and manage contacts efficiently through a menu-driven interface.

---

🚀 Features

- ➕ Add new contacts
- 📋 Display all contacts
- 🔍 Search for a contact by name
- ❌ Delete a contact

---

🛠️ Technologies Used

- C++ Programming Language
- Linked List Data Structure
- Dynamic Memory Allocation

---

⚙️ How It Works

- Contacts are stored using a linked list
- Each contact contains:
  - Name
  - Phone Number
- Users can perform operations through a simple menu system

---

▶️ How to Run the Program

Step 1: Compile

g++ contact_manager.cpp -o contact

Step 2: Run

./contact

---

💡 Concepts Used

- Structures ("struct")
- Pointers
- Linked Lists
- Functions
- Memory Management ("new" and "delete")
